# 制作 CP2K 安装包 — 操作步骤

所有脚本已准备好，你只需要在 Windows 电脑上按以下步骤操作。

---

## 第一阶段：在 Windows 上安装并测试 CP2K（约 30 分钟）

### 步骤 1：找一台 Windows 10/11 电脑

### 步骤 2：把 `scripts/1_setup_cp2k.ps1` 复制到那台电脑上

### 步骤 3：以管理员身份运行 PowerShell
- 按 `Win` 键，输入 `PowerShell`
- 右键点击 → **以管理员身份运行**

### 步骤 4：执行安装脚本
```
Set-ExecutionPolicy Bypass -Scope Process
.\1_setup_cp2k.ps1
```
> 脚本会自动安装 WSL2 和 CP2K，全程有中文提示。
> 如提示需要重启，重启后再次运行。

### 步骤 5：验证 CP2K 是否正常
```
wsl -d Ubuntu-22.04 -- cp2k --version
```
看到版本号就说明成功了。

---

## 第二阶段：制作可分发的安装包（约 20 分钟）

### 步骤 6：导出 WSL 镜像
```
.\2_export_distro.ps1
```
会生成 `cp2k_distro.tar`（约 1-2GB）

### 步骤 7：下载安装 Inno Setup
- 访问 https://jrsoftware.org/isdl.php
- 下载并安装（免费软件，一路下一步）

### 步骤 8：把整个项目文件夹复制到 Windows 上
确保目录结构如下：
```
cp2k-windows-installer/
├── cp2k_distro.tar          ← 第6步生成的文件
├── scripts/
│   ├── run_cp2k.bat
│   ├── cp2k_shell.bat
│   └── uninstall_wsl.ps1
├── installer/
│   └── cp2k_setup.iss
└── README.txt
```

### 步骤 9：用 Inno Setup 编译安装包
1. 双击打开 `installer/cp2k_setup.iss`
2. 点击菜单 **Build → Compile**（或按 F9）
3. 等待编译完成，在 `output/` 文件夹里找到 `CP2K_2024_Windows安装包.exe`

---

## 完成！

生成的 `.exe` 文件就是最终安装包，发给任何人双击即可安装使用。

---

## 常见问题

**Q：提示"虚拟化未启用"怎么办？**
A：重启电脑，进入 BIOS，找到 `Virtualization Technology` 或 `VT-x`，设为 `Enabled`

**Q：公司/学校电脑无法安装怎么办？**
A：可能被 IT 限制了 WSL，需要联系管理员开放权限

**Q：安装包多大？**
A：约 1-2GB，因为包含完整的 Linux 环境
