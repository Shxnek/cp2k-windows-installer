CP2K Quantum Chemistry Software – Windows Edition
==================================================

[How to Run a Calculation]

  Double-click the "CP2K" desktop icon to open a shell.
  The shell tries to open in your Windows user home folder automatically.
  If that path cannot be resolved inside WSL, it falls back to the Linux home folder.
  Then type:
    cp2k -i yourfile.inp


[System Requirements]

- Windows 10 (version 2004 or later) or Windows 11
- At least 4 GB RAM
- At least 6 GB disk space
- CPU virtualisation must be enabled
- Windows 10 may require the WSL2 Linux kernel update package after WSL is enabled


[How to Uninstall]

Control Panel → Programs → Uninstall a program → CP2K → Uninstall


[Get Help]

CP2K official website : https://www.cp2k.org
CP2K manual           : https://manual.cp2k.org

CP2K 量子化学计算软件 - Windows 版
====================================

【如何运行计算】

  双击桌面 "CP2K" 图标，打开命令行。
  Shell 会优先定位到你的 Windows 用户主目录。
  如果该路径在 WSL 中无法解析，则会回退到 Linux 用户主目录。
  然后输入：
    cp2k -i 你的文件.inp


【系统要求】

- Windows 10 (2004 版本或更高) 或 Windows 11
- 至少 4GB 内存
- 至少 6GB 磁盘空间
- CPU 需开启虚拟化
- Windows 10 在启用 WSL 后可能还需要安装 WSL2 Linux 内核更新包


【卸载方法】

通过"控制面板 → 程序 → 卸载程序"卸载 CP2K


【获取帮助】

CP2K 官方网站: https://www.cp2k.org
CP2K 手册:     https://manual.cp2k.org
