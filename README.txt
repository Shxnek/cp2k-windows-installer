CP2K Quantum Chemistry Software – Windows Edition
==================================================

[How to Run a Calculation]

  Double-click the "CP2K" desktop icon to open a shell.
  The shell opens in your Windows user home folder automatically.
  Then type:
    cp2k -i yourfile.inp


[System Requirements]

- Windows 10 (version 2004 or later) or Windows 11
- At least 4 GB RAM
- At least 6 GB disk space


[How to Uninstall]

Control Panel → Programs → Uninstall a program → CP2K → Uninstall


[Get Help]

CP2K official website : https://www.cp2k.org
CP2K manual           : https://manual.cp2k.org

CP2K 量子化学计算软件 - Windows 版
====================================

【如何运行计算】

  双击桌面 "CP2K" 图标，打开命令行。
  Shell 会自动定位到你的 Windows 用户主目录。
  然后输入：
    cp2k -i 你的文件.inp


【系统要求】

- Windows 10 (2004 版本或更高) 或 Windows 11
- 至少 4GB 内存
- 至少 6GB 磁盘空间


【卸载方法】

通过"控制面板 → 程序 → 卸载程序"卸载 CP2K


【获取帮助】

CP2K 官方网站: https://www.cp2k.org
CP2K 手册:     https://manual.cp2k.org
