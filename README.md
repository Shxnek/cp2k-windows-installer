# CP2K Windows 一键安装包

CP2K 是一款用于量子化学和固体物理计算的科学软件。  
本项目让你无需任何编程知识，直接在 Windows 上安装并使用 CP2K。

---

## 下载安装包

1. 点击上方的 **Actions** 标签页
2. 点击最新一次成功运行的记录（绿色勾）
3. 拉到页面底部，找到 **Artifacts**，点击下载
4. 解压 `.zip` 文件，得到 `CP2K_2024_Windows安装包.exe`
5. 双击运行，按提示一路下一步即可完成安装

> 安装大约需要 3-5 分钟，期间会自动配置环境，请耐心等待。

---

## 系统要求

| 要求 | 说明 |
|------|------|
| 操作系统 | Windows 10（2004 版本及以上）或 Windows 11 |
| 内存 | 至少 4GB |
| 硬盘空间 | 至少 10GB 可用空间 |
| 网络 | 安装时需要联网 |

不确定自己的 Windows 版本？按 `Win + R`，输入 `winver` 回车查看。

---

## 安装完成后如何使用

安装完成后，桌面会出现 **CP2K** 快捷方式。

**运行计算：**

- 方法一：把你的 `.inp` 输入文件**直接拖拽**到桌面的 CP2K 图标上
- 方法二：双击桌面图标，打开命令行后输入：
  ```
  cp2k -i 你的文件.inp
  ```

示例文件位置：`C:\CP2K\examples\`

## 常见问题

**Q：安装时提示"需要启用虚拟化"怎么办？**  
A：重启电脑，开机时按 `Del` 或 `F2` 进入 BIOS，找到 `Virtualization Technology`，改为 `Enabled`，保存重启。

**Q：安装包多大？**  
A：约 1-2 GB，包含完整的运行环境。

**Q：下载的是 .zip 文件，不是 .exe？**  
A：正常的，GitHub 会把文件打包成 .zip 方便下载。解压后里面就是 .exe 安装包。

**Q：如何卸载？**  
A：控制面板 → 程序 → 卸载程序 → 找到 CP2K → 卸载。

## 自行重新打包

如果你想自己打包（比如更新版本），操作如下：

1. Fork 本仓库到你的 GitHub 账号
2. 点击 **Actions** → **打包 CP2K Windows 安装程序**
3. 点击 **Run workflow** 按钮
4. 等待约 20-30 分钟，完成后下载安装包
